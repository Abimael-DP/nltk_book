Installing NLTK-Lite on Macintosh (OS X)

Installing Python

NLTK-Lite requires the Python scripting language interpreter to be installed on your machine.

Download the appropriate Python binary from http://undefined.org/python/MacPython-OSX-2.4.1-1.dmg to /tmp.

Mount the dmg, double click and follow the OSX Packager install instructions.

Then open a terminal session, and modify the system's linkages from the standard Python 2.3.x to the new version: 

	sudo rm /usr/bin/python
	sudo ln -s /usr/local/bin/python /usr/bin/python

Check the Python installation: 

	python -V

Installing Numarray

NLTK also requires a Python module called Numarray, which provides a range of mathematical functions.

Download the appropriate Numarray binary from http://www.pythonmac.org/packages/numarray-1.1.1-py2.4-macosx10.3.zip to /tmp.

Mount the dmg, double click and follow the OSX Packager install instructions.

Installing NLTK-Lite

Download the NLTK-Lite source distribution from http://prdownloads.sourceforge.net/nltk/nltk_lite-0.4.tar.gz?download to /tmp.

Uncompress the NLTK-Lite source distribution: 

	cd /tmp
	gunzip nltk_lite-0.4.tar.gz
	tar -xvf nltk_lite-0.4.tar

Install the NLTK-Lite mododule: 

	cd nltk_lite-0.4
	sudo python setup.py install

Installing the NLTK-Lite Data Package

Download the NLTK-Lite Data distribution from http://prdownloads.sourceforge.net/nltk/nltk_lite-corpora-0.4.zip?download to /tmp.

Uncompress the NLTK-Lite Data distribution: 

	cd /tmp
	unzip nltk_lite-corpora-0.4.zip

Create a directory to hold the NLTK-Lite corpus set. A common path for this directory is /usr/share/nltk. If you are not a system administrator, you can put this directory in your home directory (e.g. ~/nltk). 

	mkdir /usr/share/nltk

Uncompress each corpus that you'd like to use to into the NLTK corpus directory that you created, eg: 

	unzip -d /usr/share/nltk/ brown.zip

For convenience, a recursive uncompression can be performed by: 

	cd /tmp/nltk_lite-corpora-0.4
	ls | grep zip | while read i; do unzip -d /usr/share/nltk $i; done

Set an environment variable for the NLTK-Lite Data 

	export NLTK_LITE_CORPORA=/PATH/TO/DATA

For convenience this environment variable can be set in your $HOME/.bash_profile so it is loaded every time you start a terminal session: 

	echo "export NLTK_LITE_CORPORA=/PATH/TO/DATA" >> $HOME/.bash_profile

Alternatively, you can also use the function set_basedir() in the module nltk_lite.corpora: 

	[user@host ~]$ python
	Python 2.4.1 (#1, Aug  4 2005, 10:25:27)
	[GCC 3.4.3 20050227 (Red Hat 3.4.3-22.fc3)] on linux2
	Type "help", "copyright", "credits" or "license" for more information.
	>>> from nltk_lite.corpora import set_basedir
	>>> set_basedir('/PATH/TO/DATA/')
