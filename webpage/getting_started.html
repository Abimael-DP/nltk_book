Getting Started

Introduction

NLTK is a toolkit of Python modules. There are three typical ways to use NLTK:

  • Run the Python interpreter, and use NLTK interactively.
  • Create a Python program that uses NLTK, using a text editor (such as Notepad, Emacs, or vi); and execute it (either from the command line, or by double-clicking on its icon).
  • Use an integrated development enviroment, such as IDLE or Emacs' Python Mode.

How you prefer to interact with Python and NLTK is a personal choice. If you are new to Python, we suggest that you try using an integrated development enviroment, such as IDLE. IDLE is included with Python on most systems.

Using NLTK

NLTK consists of a set of Python modules, each of which defines classes and functions related to a single data structure or task. Before you can use a module, you must import its contents. The simplest way to import the contents of a module is to use the "from module import *" command. For example, to import the contents of the nltk.token module, type: 

    >>> from nltk.token import *

For more information on importing modules, and for a general introduction to NLTK, see the "Basics" tutorial. Once you have worked through the "Basics" tutorials, you may want to look at the other tutorials.

Running IDLE

IDLE is an integrated development environment, which simplifies the task of creating and debugging Python programs. On Windows, IDLE is located in the start menu ("Programs→Python 2.1→Idle"). On unix, you can run IDLE from the command line, with the command "idle".

Within IDLE, you can use the interactive interpreter; or you can create a new Python file ("File→New Window"). To execute the new Python file, use "Edit→Run script" (or Ctrl-F5). To save the new Python file, use "File→Save" (or Ctrl-S). When you save your Python file, use the extension ".py". You can open an existing Python program with "File→Open" (or Ctrl-O).
