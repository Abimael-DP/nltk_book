NLTK: Installing on Windows

Installing Python

You must have Python 2.1 or higher installed on your system to use NLTK. If Python is already installed, you can skip this step.

  1. Download the Python installer. The Python installer for Python 2.2.1 is available here; or you can check www.python.org for the most recent version. Note the location that you save the installer to. (You can download it to your desktop, and delete it after you finish installing Python.) The Python installer is also included in the windows subdirectory of the NLTK CD.
  2. Close all applications.
  3. Run the Python installer.
  4. Follow the installation instructions. By default, Python will install to "C:\Python21". Install all components.

Installing Numeric

In order to use some the text classification modules, and a few other numerically intense modules, you must install the Numeric package. Numeric is an open-source optimized matrix library for Python. If Numeric is already installed, you can skip this step.

  1. Download the Numeric installer. You can get the latest version from SourceForge. The Numeric installer is also included in the windows subdirectory of the NLTK CD.
  2. Close all applications.
  3. Run the Numeric installer.
  4. Follow the installation instructions. Make sure to use the same installation directory that you used for Python.

Installing NLTK

  1. Download the NLTK installer. You can get the latest version of NLTK from SourceForge. The NLTK installer is also included in the windows subdirectory of the NLTK CD.
  2. Run the NLTK installer.
  3. Follow the installation instructions. Make sure to use the same installation directory that you used for Python.
