Installing NLTK

1. Requirements

1.1. Python

The natural language toolkit requires Python 2.1 or higher. To check what version of Python you are running, run "python -V". Alternatively, run an interactive Python installation session, and check which version it reports. You can download the latest version of Python from the Python download page.

1.2. Other Packages

The natural language toolkit makes use of several other open source Python packages. These packages are not required to use much of Python's core functionality. However, without them you won't be able to use some features (such as graphical demos). The following packages are currently used by the natural language toolkit:

  • Tkinter ("Tk interface") is the standard Python interface to the Tk GUI toolkit. This is usually installed by default under Windows, but you may need to install it separately if you are using Unix or a Macintosh. Tkinter should be included in your version of Python. A RedHat 7.1 RPM is also available from the Python download page.
  • Numeric Python adds a fast array facility to the Python language. It can be downloaded from the Numeric Python download page.

The following are packages that we are considering using in the future:

  • PyWordNet: a python interface to WordNet. WordNet is a lexical database, developed at Princeton University. PyWordNet can be downloaded from the PyWordNet download page. The WordNet database files can be downloaded from the WordNet homepage.

2. Windows Installation

After you have installed Python, download and run the NLTK installation file (nltk-??.??.win32.exe). The installation wizard will guide you through the installation process.

2.1. Upgrading

When upgrading the version of NLTK you are using, the installation wizard will ask you if you wish to overwrite the old NLTK files. Answer "yes."

3. RPM Installation

If you are running RedHat 7.1 (or a compatible system), you can install NLTK from the RPM file:

    rpm -i nltk-??.??-1.noarch.rpm

If you wish to build an RPM for NLTK on a system other than RedHat 7.1, you can use the SRPM file:

    rpm --rebuild nltk-??.??-1.noarch.rpm

4. Unix and Macintosch (OS X) Installation

You can manually install the toolkit from the .tar.gz file:

    gunzip nltk-??.??.tar.gz 
    tar -xvf nltk-??.??.tar
    cd nltk-??.??
    python setup.py install

You can also manually install the toolkit from the .zip file:

    unzip nltk-??.??.zip
    cd nltk-??.??
    python setup.py install
