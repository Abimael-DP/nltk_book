Installing NLTK

NLTK is divided into three packages, each of which can be downloaded from the NLTK download page. The NLTK package contains the Natural Language Toolkit. The NLTK data package contains a collection of corpora and sample data that can be easily used with NLTK. The NLTK contributions package contains third-party modules and packages based on NLTK that have been contributed to the NLTK project, but not integrated into the toolkit.

Installation Requirements

The natural language toolkit requires Python 2.3 or higher. To check what version of Python you are running, run "python -V". Alternatively, run an interactive Python installation session, and check which version it reports. You can download the latest version of Python from the Python download page.

The natural language toolkit makes use of several other open source Python packages. These packages are not required to use most of Python's core functionality. However, without them you won't be able to use some features of NLTK (such as graphical demos). The following packages are currently used by the natural language toolkit:

  • Tkinter ("Tk interface"): The standard Python interface to the Tk GUI toolkit. This is usually installed by default, but you may need to install it separately. [download RPM]
  • Numeric Python: A fast array manipulation package for Python. [download Numeric]
