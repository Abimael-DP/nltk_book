NLP Toolkit Timetable

This page indicates the current and projected status of the toolkit. For this purpose, the toolkit is broken up into a hierarchy of units. Each unit can be used to teach students about a particular subject related to natural language processing.

NLTK Units

The units currently defined by the toolkit are:

  • Words: Tokens vs types, distributions of words, tokenization, zipfs law, collocations.
  • Basic tagging and language modelling: POS tagging, nth order taggers, nth order language models
  • Advanced tagging and language modelling: backoff, discounting, MLE, Laplace's law, HMMs
  • Parsing
    • Chunk parsing
    • Deterministic Chart Parsing
    • Probablistic Chart Parsing
  • Probability and Statistics
    • Probability: entropy, chi^2, information theory, etc.
    • Statistical tools: HMMs, EM, maxent
  • Semantics
    • Wordnet
    • Word Sense Disambiguation
    • Machine Translation
  • Signal processing: filters, signals, speech processing
  • Information Retrieval: TFIDF, document vectors, etc.
  • Data Structures: What data structures can be used to encode different types of linguistic data? Annotation graphs, representing traces in syntax trees, etc.

Each unit consists of one or more Python modules, tutorial-style documentation, and one or more set of problems and solutions. The current overall status of each unit is show in the following table.
