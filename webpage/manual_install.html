NLTK: Manual Install on Unix

If you wish to do a manual installation of Python or the toolkit, you're mostly on your own: there are too many variables to allow me to give good general instructions. However, the following sets of command should work, in theory:

Python and Tkinter

      gunzip Python2.1.1.tgz
      tar -xvf Python2.1.1.tar
      cd Python2.1.1
      ./configure
      make install
      ln -s /usr/local/bin/python2.1 /usr/bin/python2.1
      ln -s /usr/local/lib/python2.1 /usr/lib/python2.1
      ln -s /usr/local/include/python2.1 /usr/include/python2.1
      mv /usr/bin/python /usr/bin/python.bak
      ln -s python2.1 /usr/bin/python

IDLE

(from the Python2.1.1 directory)

      cp -R Tools/idle/ /usr/local/lib/python2.1/site-packages/

To run idle, type:

      % python2 /usr/local/lib/python2.1/site-packages/idle/idle.py

Numeric

      gunzip Numeric-20.1.0.tar.gz 
      tar -xvf Numeric-20.1.0.tar
      cd Numeric-20.1.0
      python2.1 setup.py install

NLTK

      gunzip nltk-1.0.tar.gz 
      tar -xvf nltk-1.0.tar
      cd nltk-1.0
      python2.1 setup.py install
