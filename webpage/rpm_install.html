NLTK: Installing on Unix with RPMs

Python

First, you'll want to install Python and Tkinter:

    % rpm -i python2-2.1.1-3.i386.rpm python2-tkinter-2.1.1-3.i386.rpm

If you have any troubles with this step, and you can't resolve them yourself, you might be better off trying the SRPM install or manual install instructions (below).

Different RPMs have different ideas about where Python is installed. The following commands make symbolic links, so that RPMs and other tools are less likely to get confused.

    % ln -s /usr/bin/python2.1 /usr/local/bin/python2.1
    % ln -s python2.1 /usr/local/bin/python
    % ln -s /usr/lib/python2.1 /usr/local/lib/python2.1
    % ln -s /usr/include/python2.1 /usr/local/include/python2.1

If you want "python" to run Python 2.1, instead of Python 1.5, use the following commands:

    % mv /usr/bin/python /usr/bin/python.bak
    % ln -s python2.1 /usr/bin/python

Note that the old version of Python can still be accessed via "/usr/bin/python1.5".

NLTK and Support Files

Install the Natural Langauge Toolkit, and packages that it depends on:

    % rpm -i Numeric-20.1.0-1.i386.rpm
    % echo Numeric >/usr/lib/python2.1/site-packages/Numeric.pth
    % rpm -i nltk-1.0-1.noarch.rpm

Numeric implements fast, compact, multidementional array support.

IDLE

If you want to use IDLE (an integrated development environment for Python), install the Python tools RPM: 

  
    % rpm -i python2-tools-2.1.1-3.i386.rpm

Unfortunately, this RPM conflicts with the Python 1.5 tools RPM. If you installed the Python 1.5 tools RPM, you will get the following error message: 

    % rpm -i python2-tools-2.1.1-3.i386.rpm
    file /usr/bin/idle from install of python2-tools-2.1.1-3 
    conflicts with file from package python-tools-1.5.2-30

Luckily, it's a trivial conflict (the file in question is a one-line shellscript). Tell rpm to ignore the conflict:

    % rpm -i --replacefiles python2-tools-2.1.1-3.i386.rpm

Python Development

If you think you might write C extensions to Python, install the Python development RPM:

    % rpm -i python2-devel-2.1.1-3.i386.rpm

Uninstalling

If for some reason you need to uninstall, you can do so with "rpm -e":

      rpm -e python2-devel-2.1.1-3
      rpm -e nltk-1.0-1
      rpm -e Numeric-20.1.0-1
      rpm -e python2-tools-2.1.1-3
      rpm -e python2-tkinter-2.1.1-3
      rpm -e python2-2.1.1-3
