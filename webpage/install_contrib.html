Installing the NLTK Contributions Package

2. Windows Installation

Download and run the NLTK contributions package installation file (nltk_contrib-??.??.win32.exe). The installation wizard will guide you through the installation process.

2.1. Upgrading

When upgrading the version of the NLTK corpus package you are using, the installation wizard will ask you if you wish to overwrite the old files. Answer "yes."

3. RPM Installation

If you are running RedHat 7.1 (or a compatible system), you can install the NLTK contributions package from the RPM file:

    rpm -i nltk_contrib-??.??-1.noarch.rpm

If you wish to build an RPM for the NLTK contributions package on a system other than RedHat 7.1, you can use the SRPM file:

    rpm --rebuild nltk_contrib-??.??-1.noarch.rpm

4. Unix and Macintosch (OS X) Installation

You can manually install the NLTK contributions package from the .tar.gz file:

    gunzip nltk_contrib-??.??.tar.gz 
    tar -xvf nltk_contrib-??.??.tar
    cd nltk_contrib-??.??
    python setup.py install

You can also manually install the contributions package from the .zip file:

    unzip nltk_contrib-??.??.zip
    cd nltk_contrib-??.??
    python setup.py install
